Table element I've learnt
Element Description
The Root Element
table Which is main container.
td used to create data cells.
tr represents the row.
Table Heading
th add a table heading.
thead used to create a seperate header for a table.
Other Attribute
Scope identifies whether a cell is a header for column,row,or group of columns or rows.
caption provide a description about the data presented in a table.
colspan & rowspan stretches two or more calls across a table.
tbody it indicates the main body of the table.
thead used to create seperate footer for the table.
